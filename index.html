Conditional and looping:

Testing to see if a condition is true or false, and cycling through a set of instructions until some condition is met.

Variables and data structures:

storing information, which may change over time And many more to come!

Javascript (including ES6-ES2021)

You will learn new JavaScript standards from ES6-ES2021. we will see all the changes happens to JS starting to latest

Functional Programming

It is a declarative type of programming style, Functional programming is a programming paradigm in which we try to bind everything in pure functions style.

Object Oriented Programming

Object-oriented programming (OOP) is a computer programming model that organizes software design around data, or objects, rather than functions and logic

Scope and Execution Context

scope is function-based while context is object-based.scope pertains to the variable access of a function when it is invoked and is unique to each invocation. Context is always the value of the this keyword And many more to come!

Asynchronous JavaScript + Event Loop

The event loop is the core of every asyncio application. Event loops run asynchronous tasks and callbacks, perform network IO operations, and run subprocesses.

JavaScript Engine and Runtime

A JavaScript engine is a program or interpreter which reads our JavaScript code, produces machine code, and finally runs the machine code. It is landed in JavaScript runtimes like web browsers, Node. js

Stack Overflow & Memory Leaks

whenever you allocate memory with malloc/new and don't deallocate it later with free/delete after finishing using that memory... will cause a memory leak! The allocated memory will remain there and that space won't be used by your program ever again.

Higher Order Functions

A higher order function is a function that takes a function as an argument, or returns a function And many more to come!
